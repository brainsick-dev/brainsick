# Brainsick

An open source compiler for the classic Visual Basic (VB/VB6) language 

## About

`bscc` is the command line compiler driver.

## Status

A prototype, capable of compiling Hello World.

Expect the git history to be continuously rebased, until things are more mature.

## Contributing

Thanks for your interest. However, due to the prototype status, contributions are unlikely to be helpful.
